WINDOWS USERS - IMPORTANT:
  When you install Git for Windows, you get "Git Bash" - a terminal
  that lets you run the commands below. After installing Git, search
  for "Git Bash" in your Start menu and use that (not Command Prompt).

  (Just click Next through all the options - defaults are fine)

  Install Python 3: https://www.python.org/downloads/
  (IMPORTANT: Check "Add Python to PATH" during installation)

Mac:
  Install via Xcode Command Line Tools (includes git):
  xcode-select --install

  Or install both via Homebrew:
  brew install git python3

Linux:
  sudo apt-get install git python3 python3-pip
