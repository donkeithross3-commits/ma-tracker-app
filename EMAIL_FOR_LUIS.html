Step 1:

Then copy-paste these commands ONE AT A TIME:

‚ö†Ô∏è IMPORTANT: Copy and paste each line ONE AT A TIME, pressing Enter after each.

Step 2:
Get ngrok token (one-time, takes 1 minute):

1. Sign Up for ngrok (Free)
2. Get Your Auth Token

Then run (paste your token):
~/bin/ngrok config add-authtoken YOUR_TOKEN_HERE
Step 3:
Configure IB Gateway (you probably already have this):
  • Open IB Gateway/TWS
  • Go to: Settings ‚Üí API ‚Üí Settings
  • Enable: "Enable ActiveX and Socket Clients"
  • Port: 7497 (paper) or 7496 (live)
  • Trusted IPs: Add 127.0.0.1
  • Click OK
Step 4:
Start the scanner:
~/start-scanner.sh

Done! You'll see your public URL. Send that URL to the team!
